## Setup

Install dependencies

```
yarn install
```

## Run

### React Development in Browser

```
yarn dev
```

## Production Build (For Cordova)

```
yarn build
```

## Testing

All tests can be run continually during development:

```
yarn dev:test
```

Or just one for CI etc.

```
yarn test
```
